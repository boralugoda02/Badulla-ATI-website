## Overview
A web-based portal designed for **ATI Badulla**. It provides an easy-to-use interface for students and staff to access institute information and administrative tools.

## Key Features
* **Clean Homepage**: A professional introduction to the institute.
* **Exam Announcements**: Timely updates on upcoming examinations.
* **Live Visitor Counter**: Real-time tracking of website traffic.
* **Professional Course Catalog**:
    * HNDIT (Higher National Diploma in Information Technology)
    * HNDE (Higher National Diploma in Engineering)
    * HNDA (Higher National Diploma in Accountancy)
* **Secure Admin Portal**: A protected login area for staff to manage institute operations.

## Technical Details
* **Development IDE**: NetBeans
* **Environment**: Runs on `localhost` (requires Apache Tomcat)

## How to Run
1. Ensure you have **Apache Tomcat** installed.
2. Import the project into your **NetBeans IDE**.
3. Configure the server and deploy the application.
4. Run the project to view it on your local browser.
